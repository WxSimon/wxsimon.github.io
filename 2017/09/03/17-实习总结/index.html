实习总结

Posted on 2017-09-03 Edited on 2025-10-15 In 总结

时间：2017.7.3 — 2017.9.3

刚好两个月的时间，我觉得有必要对我的第一次实习做一个总结。

从第一天说起

6 月 30 号。
因为前天晚上我和室友们一起打麻将到很晚，早上起来的时候正好下着一场大雨。
那场雨真的是一点面子也不给，我们四个冒着雨从宾馆冲回宿舍。

因为东西都已经收拾得差不多了，拿上行李箱后，我们便赶去接我们的那辆大巴。
四个人中，三个人去一个地方，而我一个人要去另一个地方。
庆幸的是，我们乘坐的是同一辆车。

伴随着忽大忽小的雨，我们到达了武汉。
可是一到武汉，我整个人的抗拒感就更强了。
我抗拒的不是这座城市，而是即将去的那个“培训机构”。
我讨厌去学那些自己丝毫不感兴趣的东西，
我讨厌学校的领导把我们当作他们赚钱的工具。

好在，我是幸运的。
我成功“逃离”了那个地方，去了另一个能继续学习我喜欢知识的地方。

那天也发生了一些小插曲，比如我怕纯纯等得太久，就自己离开了集合点，结果被班主任小小批评了一顿。
之后还不得不去面对人生第一次租房。
拖着巨大的行李箱，独自住进了这座陌生的城市。

两个月中

第一天在公司上完班之后，心里的恐惧感少了几分，取而代之的是慢慢适应。
适应武汉的物价，适应和学校不同的生活方式，也逐渐适应了职场节奏。

两个月的时间里，我掌握或接触到了很多新的技术栈：
• Spring Boot
• Redis
• HBase
• Vue
• 前端基础
• ELK
• 分布式服务
• Nginx
• Maven

这些几乎都是我在学校没接触过的技术。
在这两个月里，我每天都和它们打交道。
最大的感受是：学校里学的东西和公司里用的东西真的完全不一样。

学校还在用“三大框架”和各种 XML 配置，
而学习了 Spring Boot 之后，我发现它能大大简化开发流程、提升效率。

因为 Boss 的要求，我这个前端小白还硬着头皮把一个前端项目迁移成了 Vue 项目。
在迁移的过程中，我也学到了不少前端知识，包括 Webpack、jQuery、Bootstrap 等等。

之后

实习结束后，我打算好好整理一下自己的简历。
秋招，我来了！！！ 💪
