Java 项目部署

环境依赖

1、jdk
2、maven
3、ssh

操作过程

1 创建项目(省略)

2 添加 workflow 文件

在项目的根目录下，创建 .github/workflows 文件夹
新建 build.yml 文件

3、编写服务流程

其中 REMOTE_HOST、REMOTE_USER、SSH_PRIVATE_KEY 为服务器相关配置，使用 github 的secrets 配置。[项目]->[setting]->[secrets] 进行配置
